# 🏦 СергоБанк - Шуточный Telegram бот для возврата долгов

Шуточно-юридический Telegram бот, который напоминает твоему другу Степану о долге в 4000 рублей!

## 🎯 Возможности бота:

- **Напоминания каждые 4 часа** с матерными фразами
- **Начисление процентов** 2% в месяц с 1 августа
- **Профиль должника** с историей платежей
- **Голосовые сообщения** от имени банка
- **Ответы на сообщения** с угарными фразами
- **Команды** для управления долгом

## 📋 Команды:

- `/start` - Начало работы
- `/status` - Статус долга
- `/pay` - Оплата долга
- `/profile` - Профиль должника
- `/voice` - Отправить голосовое сообщение

## 🚀 Установка:

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Скопируй `.env.example` в `.env` и заполни:
```bash
cp .env.example .env
```

3. Заполни `.env` файл:
```
BOT_TOKEN=твой_токен_от_BotFather
STEPAN_ID=telegram_id_степана
SERGEY_ID=твой_telegram_id
```

4. Запусти бота:
```bash
python bot.py
```

## ⚙️ Настройка:

Все настройки находятся в `config.py`:
- Начальная сумма долга
- Процентная ставка
- Интервал напоминаний
- Имена участников

## 📊 Как это работает:

1. Бот напоминает Степану каждые 4 часа о долге
2. С 1 августа начинают начисляться проценты 2% в месяц
3. Степан может "оплатить" долг (имитация)
4. Все операции записываются в файл `sergobank_data.json`

## 🎭 Матерные фразы:

Бот использует тяжёлый мат для максимального угараста! Все фразы шуточные, но убедительные.

## ⚠️ Внимание:

Это шуточное приложение! Не используйте его для реальных финансовых операций!
